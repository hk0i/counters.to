## Core Playbook
Mizuki throws Spirit Glaive and Binding Chain to disrupt enemy positioning, using Katashiro Return to reposition aggressively while Remedy Aura keeps nearby allies topped off during the fight.

## Tactical Caveats
Remedy Aura's healing scales with dealing damage, so if she's forced to play passively and never lands hits, her sustain output falls well behind a dedicated healer.
